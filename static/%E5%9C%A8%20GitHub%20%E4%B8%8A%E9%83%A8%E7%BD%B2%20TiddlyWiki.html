在 GitHub 上部署 TiddlyWiki

 2022年06月13日 10:56

目标

通过此文章，你将达成如下目标：

  1. 所有的 wiki 内容均存储在 GitHub 中
  2. 可通过 GitHub Pages 在任何地方访问你的 TiddlyWiki 页面

GitHub 步骤

  1. 在 GitHub 中创建一个空白 Repo，比如 qgymib/wiki
  2. 将 Repo Clone 到本地
  3. 在 Repo 中执行如下指令：
    npm install -s tiddlywiki
  4. 新建 .gitignore 文件并写入如下内容：
    # Ignore MacOS auto generated folder
.DS_Store

# Ignore Windows thumbnails
Thumbs.db

# Ignore nodejs modules
node_modules/

# Ignore tiddlywiki build output files
/wiki/output

# Always ignore import informations
/wiki/tiddlers/$__Import.tid
  5. 新建 .github/workflows/deploy.yml 文件并写入如下内容：
    name: deploy

on:
  push:
    branches: [ "master" ]

  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v3

      - name: Setup Node.js environment
        uses: actions/setup-node@v3

      - name: Setup TiddlyWiki
        run: npm install

      - name: Build static files
        run: npx tiddlywiki wiki --build

      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./wiki/output
          force_orphan: true
  6. 将上述内容提交上传至 GitHub

注意事项

  1. $__StoryList.tid 文件的内容应总是保持为：
    list: GettingStarted
title: $:/StoryList
  2. $__Import.tid 应总是被忽略
